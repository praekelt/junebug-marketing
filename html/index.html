Local mobile network integration made easy

Junebug is a messaging gateway. This new developer tool simplifies integration with local mobile networks. Now governments, NGOs and enterprises can send and receive millions of SMS, USSD and voice messages faster and better than ever before.

Simpler Integration

Junebug makes life simpler for operators and application developers. It integrates messaging through a single API, exposing Junebugâ€™s capabilities in a standardised message format.

Local Installation

Junebug was designed for local installations. This speeds up and improves efficiency when you send, receive and process messages. Local installation also means that Junebug guarantees data sovereignty and specific country laws can be adhered to.

Direct Integration

Junebug lets application developers integrate directly with local mobile network infrastructure. This shortcuts rerouting via a mobile integrator. Direct integration also gives developers full access to mobile network data. This insight helps isolate and solve problems quickly.

Junebug Beta lets you

  • Create, update, delete and list integrations to aggregators or MNOs.
  • Send and receive SMS, USSD and Voice messages.
  • Receive updates on message delivery status.
  • Query the health and status of each integration.
  • Query the number of MO and MT messages and delivery events being sent and received over each integration.
  • Retrieve basic low-level logging messages for each integration.
